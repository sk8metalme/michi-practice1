# タスク分割: Web電卓アプリケーション

> **プロジェクト**: michi-practice1  
> **作成日**: 2025-11-06  
> **Epic**: MP-1  
> **期限**: 2025-11-30  
> **総見積もり**: 52 SP（≈ 26人日）

## 概要

Java + Spring Boot + Thymeleafを使用した、Onion ArchitectureベースのWeb電卓アプリケーション。TDD（テスト駆動開発）で実装し、四則演算と計算履歴機能を提供します。

**技術スタック**:
- 言語: Java 17
- フレームワーク: Spring Boot 3.x
- フロントエンド: Thymeleaf + Bootstrap
- アーキテクチャ: Onion Architecture
- 開発手法: TDD（テスト駆動開発）
- テストカバレッジ目標: 95%以上
